# 💰 Money Quest

RPG形式でお金の知識を冒険しながら学ぶ、金融教育ノベルゲームです。

子どもが楽しみながら金融リテラシーを身につけられる設計ですが、大人が見ても刺さる実用的な金融知識を盛り込んでいます。

## 🎮 プレイ方法

`index.html` をブラウザで開くだけで遊べます。サーバー不要・インストール不要。

## 📖 ゲーム概要

ドラクエ風のクエスト構造で「お金の冒険」を進めながら、現実の金融知識を自然に習得できます。

### 4つのエリア（＝ゲームの4大エリア）

| エリア | 学習テーマ | ラスボス |
|---|---|---|
| 貯める | 先取り貯蓄・複利・72の法則 | 衝動買いの魔王 |
| 稼ぐ | 価値提供・副収入の多様性 | 時間切り売りの呪い |
| 使う | 消費・投資・浪費の違い | 広告の罠 |
| 守る | 詐欺・ポンジスキーム対策 | 甘い話の悪魔 |

### ゲームの流れ

1. **シナリオ**：選択肢によって所持金が変動
2. **ボス戦**：金融知識の理解テスト（クイズ形式）
3. **クリア**：習得した知識の確認

## 🎯 ターゲット

- 子ども（8〜14歳）：ゲームとして楽しむ
- 親・大人：一緒にプレイしながら金融を考えるきっかけに
