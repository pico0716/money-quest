## 収益化アイデア（精査済み）

### money-quest 単体
| 案 | 評価 | コメント |
|---|---|---|
| ゲーム自体を有料販売 | △ | 個人・無名の有料DLは売れにくい。まず無料で母数 |
| 完全無料＋アフィリエイト導線 | ◎ | 本命。子どもが遊ぶ→親が解説リンクを踏む→送客 |
| 親向け「解説書」をnote有料販売 | ○ | ゲームは無料、深掘りは有料。教育コンテンツとして自然 |
| 寄付/応援（投げ銭） | △ | 補助的。コンテンツ実績ができてから |

**結論：ゲームは無料。収益はアフィリエイト＋note有料コンテンツで取る。**

### 連携収益導線（核）
```
子どもが money-quest で遊ぶ（無料・拡散）
    ↓ ゲーム内「大人はこっち」リンク
親が investment-quiz で診断（自分ごと化）
    ↓ 診断結果ページのアフィリエイト
証券口座開設・金融サービス（ASP報酬）
    ↓ より深く知りたい層
note 有料記事・マガジン（ストック収益）
```
- 子ども向け無料ゲームが「広告塔」、大人が収益化ポイント。
- ターゲットを「大人本命」にした理由がここで効く。

### ASP承認待ち期間にできる先行準備（重要）
1. money-quest MVP（エリア1）実装 → 発信ネタを先に作る
2. note 記事を5〜10本ストック（公開タイミングを揃える）
3. X で「金融教育ゲームを子どものために作ってる」発信を先行スタート（承認前から認知を貯める）
